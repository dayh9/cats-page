# cats-page

### Vue + vuetify application using the cat api

`https://thecatapi.com/`

You can replace `"DEMO-API-KEY"` in **RandomCat** and
**CatsGallery** views

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

